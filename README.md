Indian Gov Scheme Eligibility Checker
=====================================

Scaffold for the Scheme Eligibility Checker project. Goals:

- Convert the provided blueprint into concrete requirements and rules.
- Implement a small, testable eligibility engine in Python.
- Add CLI, tests, and CI so we can iterate and push small changes.

Next steps:

- Provide the full `Scheme_Checker_Project_Blueprint.docx` file so I can extract a Markdown summary and convert requirements into rules.
- Iterate: implement rules, expand tests, and push commits to your GitHub remote.

Quick start

1. Create a Python virtualenv and install dev deps:

```bash
python -m venv .venv
.venv\Scripts\activate    # Windows
pip install -r requirements.txt
```

2. Run tests:

```bash
pytest -q
```
